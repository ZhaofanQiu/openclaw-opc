### 修复详情

#### 1. 后端 API 修复 (`packages/opc-core/src/opc_core/api/budget.py`)

**问题**: `get_budget_stats()` 返回的字段名与 API 期望的不匹配

**修复前**:
```python
return {
    "total_budget": stats["total_budget"],
    "total_used": stats["total_used"],
    "total_remaining": stats["total_remaining"],  # 不存在！
    "employee_count": stats["employee_count"],      # 不存在！
    "avg_budget": stats["avg_budget"],              # 不存在！
    "avg_remaining": stats["avg_remaining"],
}
```

**修复后**:
```python
return {
    "total_budget": stats["total_budget"],
    "total_used": stats["total_used"],
    "total_remaining": stats["total_budget"] - stats["total_used"],
    "employee_count": stats["total_employees"],
    "avg_budget": stats["total_budget"] / max(stats["total_employees"], 1),
    "avg_remaining": stats["avg_remaining"],
}
```

#### 2. 分析页面 JS 修复 (`packages/opc-ui/src/views/WorkflowAnalyticsView.vue`)

**问题**: 表格 slot 使用 `{ row }` 解构，当数据为空时导致 `Cannot destructure property 'row' of 'undefined'`

**修复**: 改为使用 `scope` 对象并添加空值检查
```vue
<template #default="scope">
  <el-tag type="success" v-if="scope?.row">
    {{ scope.row.completed_count || 0 }}
  </el-tag>
</template>
```
