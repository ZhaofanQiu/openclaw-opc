## 🎨 设计特性验证

### 级别差异化

| 级别 | 名称 | 特征 |
|------|------|------|
| 1 | 实习生 | 基础人形 + 职业道具 |
| 2 | 专员 | 基础人形 + 职业道具 |
| 3 | 资深 | 领带/蝴蝶结 + 银色 S 徽章 |
| 4 | 专家 | 眼镜 + 金色 E 徽章 |
| 5 | 合伙人 | 光环 + 皇冠 + 红披风 + P 徽章 |

### 配色方案

```javascript
// 每个级别4套配色，根据员工ID哈希确定
intern:      body: ['#E6F3FF', '#FFF0E6', '#F0E6FF', '#E6FFE6']
specialist:  body: ['#4169E1', '#2E8B57', '#6A5ACD', '#708090']
senior:      body: ['#191970', '#006400', '#8B008B', '#483D8B']
expert:      body: ['#4B0082', '#8B4513', '#2F4F4F', '#800080']
partner:     body: ['#8B0000', '#DC143C', '#B22222', '#CD5C5C']
```

### 性别推断
- 英文名识别: Alice/Carol → female, Bob/Dave → male
- 回退策略: ID 哈希奇偶性决定
