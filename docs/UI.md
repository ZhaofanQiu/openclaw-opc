# 界面设计规范 (UI Design)

## 设计原则

1. **清晰优先** - 复杂功能简单呈现
2. **即时反馈** - 操作有响应，状态实时更新
3. **温度感** - 像素风营造办公室氛围
4. **可控感** - 预算、状态随时可见

## 色彩系统

### 主色调

```css
/* 主色调 */
--color-primary: #1a1a2e;        /* 深蓝 - 专业沉稳 */
--color-secondary: #00d4aa;      /* 青色 - 科技感 */
--color-accent: #ff6b35;         /* 橙色 - 警告强调 */

/* 背景色 */
--color-bg: #f5f5f7;             /* 浅灰 - 舒适 */
--color-card: #ffffff;           /* 纯白 - 卡片 */

/* 文字色 */
--color-text-primary: #1a1a2e;
--color-text-secondary: #666666;
--color-text-muted: #999999;

/* 状态色 */
--color-success: #00d4aa;
--color-warning: #ffc107;
--color-danger: #ff6b35;
--color-info: #2196f3;
```

### 暗黑模式

```css
/* 暗黑模式 */
--color-bg: #0f0f23;
--color-card: #1a1a2e;
--color-text-primary: #e0e0e0;
--color-text-secondary: #a0a0a0;
```

## 组件规范

### 按钮

```
主要按钮:    [确认]       背景: 青色, 文字: 白色
次要按钮:    [取消]       背景: 灰色, 文字: 深色
危险按钮:    [删除]       背景: 橙色, 文字: 白色
文字按钮:    [查看详情]   无边框, 青色文字
```

### 卡片

```
圆角: 8px
阴影: 0 2px 8px rgba(0,0,0,0.08)
边框: 1px solid rgba(0,0,0,0.05)
内边距: 16px
```

### 进度条

```
高度: 8px
圆角: 4px
颜色: 绿色(正常) / 黄色(警告) / 红色(危险)
```

## 动画规范

### 像素办公室动画

```
员工状态切换: 200ms ease-out
员工走动: 500ms linear (循环)
气泡弹出: 150ms bounce
点击反馈: 100ms scale(0.95)
```

### 页面过渡

```
页面切换: 300ms fade
模态框: 200ms slide-up
通知: 300ms slide-in
```

## 技术实现建议

| 部分 | 技术 | 原因 |
|------|------|------|
| 框架 | React 18 | 生态成熟 |
| 构建 | Vite | 快速开发 |
| 状态 | Zustand | 轻量够用 |
| 样式 | Tailwind CSS | 快速迭代 |
| 图表 | Recharts | 简单易用 |
| 像素 | PixiJS | 性能好 |
