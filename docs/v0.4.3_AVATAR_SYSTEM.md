# v0.4.3 员工头像系统方案

## 版本信息
- **版本号**: v0.4.3
- **发布类型**: 问题修复
- **主要目标**: 解决 emoji 头像在某些环境下显示为空白方块的问题

---

## 问题背景

当前系统使用 emoji 作为员工头像，但在以下环境存在问题：
- 缺少 emoji 字体的系统
- Linux 服务器环境
- 某些浏览器配置

导致员工头像显示为空白方块或乱码，影响用户体验。

---

## 解决方案

采用**方案一: SVG 图标映射**，实现纯前端的矢量头像系统。

### 核心设计

1. **SVG 矢量渲染**
   - 使用 80x80 viewBox 的 SVG 绘制
   - 三种形象类型：`basic` | `advanced` | `partner`
   - 性别差异：男/女发型和配饰不同

2. **差异化设计**

| 维度 | 设计 |
|------|------|
| **级别** | 5级(合伙人)有特殊光环、皇冠、披风 |
| **级别** | 4级(专家)显示金色 E 徽章和眼镜 |
| **级别** | 3级(资深)显示银色 S 徽章 |
| **级别** | 1-2级显示职业道具(试管/图表/放大镜) |
| **性别** | 女性：长发、蝴蝶结；男性：短发、领带 |
| **颜色** | 每个员工根据 ID 哈希生成唯一的配色 |

3. **配色方案**

## 技术细节

### 性别推断
```javascript
const gender = computed(() => {
  const name = props.employee.name || ''
  // 常见英文名推断
  if (name.includes('Alice') || name.includes('Carol')) return 'female'
  if (name.includes('Bob') || name.includes('Dave')) return 'male'
  // 根据 ID 哈希随机
  return props.employee.id?.charCodeAt(0) % 2 === 0 ? 'female' : 'male'
})
```

### 颜色选择算法
```javascript
const colors = computed(() => {
  const id = props.employee.id || '0'
  const hash = id.split('').reduce((acc, char) => acc + char.charCodeAt(0), 0)
  
  const scheme = colorSchemes[levelKey]
  return {
    hair: scheme.hair[hash % scheme.hair.length],
    body: scheme.body[hash % scheme.body.length],
    skin: scheme.skin[hash % scheme.skin.length],
  }
})
```

## 测试要点

1. **不同级别显示正确**
   - 实习生(1级)显示基础形象+职业道具
   - 资深(3级)显示领带/蝴蝶结+S徽章
   - 专家(4级)显示眼镜+E徽章
   - 合伙人(5级)显示光环+皇冠+披风

2. **性别差异**
   - 男性显示短发+领带
   - 女性显示长发+蝴蝶结

3. **颜色唯一性**
   - 不同员工应显示不同颜色组合
   - 同一员工始终显示相同颜色

4. **尺寸适配**
   - small(40px): 表格内显示
   - medium(64px): 卡片内显示
   - large(96px): 详情页显示
