## 问题清单

### P0 - 阻塞问题 (必须修复)

#### #1 Partner 员工缺失导致功能异常
**状态**: 🔴 待修复  
**影响**: Partner 相关功能完全不可用

**问题描述**:
- 数据库中没有 Partner 员工
- 无法唤醒 Partner
- Partner 相关功能（任务协助、管理建议等）不可用

**复现步骤**:
1. 进入 Dashboard
2. 尝试与 Partner 对话 → 无响应
3. 尝试创建需要 Partner 协助的任务 → 失败

**建议方案**:
- 选项 A: 系统初始化时自动创建默认 Partner
- 选项 B: 首次使用时引导用户创建/绑定 Partner
- 选项 C: 检查到无 Partner 时显示创建引导界面

**决策**: 采用 **选项 C** - 无侵入式引导，用户自主选择

### P2 - 中优先级

#### #5 Agent 绑定流程重复
**状态**: 🟡 待优化  
**影响**: 用户体验困惑

**问题描述**:
- 未绑定 Agent 的员工有"创建 Agent"和"绑定 Agent"两个选项
- "绑定 Agent"弹窗内又有"创建 Agent"选项
- 逻辑重复，用户困惑

**当前流程**:
```
员工卡片
  ├── 创建 Agent
  └── 绑定 Agent
        └── 弹窗内有"创建 Agent"
```

**建议方案**:
```
简化流程：
员工卡片
  └── 绑定 Agent
        ├── 选择现有 Agent (列表)
        └── 创建新 Agent (表单)
```

**具体改动**:
1. 移除员工卡片的"创建 Agent"按钮
2. 绑定弹窗使用 Tab 切换：选择现有 / 创建新
3. 创建新 Agent 后直接完成绑定

#### #6 未绑定 Agent 员工视觉状态不明确
**状态**: 🟡 待设计  
**影响**: 用户无法直观判断员工可用状态

**问题描述**:
- 未绑定 Agent 的员工也是绿色呼吸特效
- 视觉上与正常工作员工无区别
- 用户无法快速识别哪些员工"无法工作"

**当前状态**:
| 状态 | 视觉效果 |
|------|----------|
| 已绑定+在线 | 绿色呼吸 |
| 已绑定+离线 | 灰色 |
| 未绑定 | 绿色呼吸 ❌ |

**建议方案**:
```
未绑定 Agent 员工：
- 头像：灰度/半透明
- 边框：虚线或灰色
- 呼吸特效：停止或改为灰色闪烁
- 添加"未绑定"标签覆盖在头像上
```

**视觉设计**:
```css

  opacity: 0.7;
  filter: grayscale(80%);
}
.employee-card.unbound .avatar {
  border: 2px dashed #999;
}
.employee-card.unbound::after {
  content: "未绑定";
  position: absolute;
  background: rgba(0,0,0,0.6);
  color: white;
  padding: 2px 8px;
  border-radius: 4px;
}
```

## 决策记录

### 关于 Partner 默认创建
**讨论**: 是否系统强制创建默认 Partner？

**结论**: 不强制创建，采用引导式方案
- 尊重用户选择权
- 避免创建无用的默认数据
- 引导用户理解 Partner 角色后再创建

**实现方式**:
- Dashboard 检测无 Partner 时显示引导卡片
- 首次创建普通员工时提示"是否同时创建 Partner 助理？"
- Partner 管理页面显示创建入口

### 关于月度预算
**讨论**: 总预算 vs 月度预算？

**结论**: 改为月度预算 + 手动追加模式
- 更符合 SaaS 产品计费习惯
- 允许突发情况追加预算
- 保留历史预算使用记录
