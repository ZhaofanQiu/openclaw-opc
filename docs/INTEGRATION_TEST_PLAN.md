## 第二轮：用户视角自动化测试

### 场景 1: 创建员工完整流程
1. 进入Dashboard
2. 点击"创建Agent"
3. 填写职位和技能
4. 确认创建
5. 验证Agent出现在列表

### 场景 2: 发布任务并分配
1. 点击"新建任务"
2. 填写任务信息
3. 提交任务
4. 手动分配给Agent
5. 验证Agent开始工作

### 场景 3: 预算熔断处理
1. 创建低预算Agent
2. 分配高成本任务
3. 等待预算耗尽
4. 选择熔断处理方式
5. 验证系统恢复

### 场景 4: 查看报告
1. 切换不同时间范围
2. 查看图表变化
3. 导出数据

### 场景 5: 多语言切换
1. 默认中文界面
2. 切换到英文
3. 检查各区域翻译
4. 切回中文
