## 🎯 核心目标

基于用户反馈，v0.3.0聚焦三大核心需求：

1. **🤖 自动Agent管理** - Partner自动创建/删除OpenClaw Agent，无需手动配置

3. **⚡ 架构升级** - PostgreSQL + 精确Token统计，支撑扩展

### Week 3: API Key认证

#### 目标
实现API Key认证系统，支持分级权限。

#### 技术方案

**API Key模型**
```python
class APIKey(Base):
    """API Key模型"""
    __tablename__ = "api_keys"
    
    id = Column(String, primary_key=True)
    name = Column(String)  # 显示名称
    key_hash = Column(String)  # 密钥哈希
    permissions = Column(String)  # read/write/admin
    created_at = Column(DateTime)
    expires_at = Column(DateTime, nullable=True)
    last_used_at = Column(DateTime, nullable=True)
    is_active = Column(Boolean, default=True)
```

**认证中间件**
```python
@app.middleware("http")
async def api_key_auth(request: Request, call_next):
    # 排除公开端点
    if request.url.path in PUBLIC_ENDPOINTS:
        return await call_next(request)
    
    # 获取API Key
    api_key = request.headers.get("X-API-Key")
    if not api_key:
        return JSONResponse(status_code=401, content={"detail": "API Key required"})
    
    # 验证Key
    key_record = validate_api_key(api_key)
    if not key_record:
        return JSONResponse(status_code=403, content={"detail": "Invalid API Key"})
    
    # 检查权限
    if not has_permission(key_record, request.method, request.url.path):
        return JSONResponse(status_code=403, content={"detail": "Insufficient permissions"})
    
    # 记录使用
    log_api_key_usage(key_record)
    
    return await call_next(request)
```

### Week 4: 分享链接与HTTPS

#### 目标
实现JWT签名分享链接，以及HTTPS部署支持。

#### 技术方案

**分享链接生成**
```python
def generate_share_link(path: str, expires_hours: int = 24) -> str:
    """生成带签名的分享链接"""
    payload = {
        "path": path,
        "exp": datetime.utcnow() + timedelta(hours=expires_hours),
        "nonce": secrets.token_hex(8)
    }
    token = jwt.encode(payload, SECRET_KEY, algorithm="HS256")
    return f"https://opc.example.com/share?token={token}"

def verify_share_token(token: str) -> dict:
    """验证分享链接"""
    try:
        payload = jwt.decode(token, SECRET_KEY, algorithms=["HS256"])
        return {"valid": True, "path": payload["path"]}
    except jwt.ExpiredSignatureError:
        return {"valid": False, "error": "Link expired"}
    except jwt.InvalidTokenError:
        return {"valid": False, "error": "Invalid token"}
```
