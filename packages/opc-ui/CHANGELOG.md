# opc-ui 变更日志

所有 opc-ui 模块的变更记录。

## [0.4.0] - 2026-03-24

### 新增

- 初始化模块
- Vue 3 + Vite 项目结构
- 完整的路由配置
- Pinia 状态管理
- Vue I18n 国际化 (zh-CN/en-US)

### 页面

- **Login**: API Key 登录
- **Dashboard**: 数据概览统计
- **Employees**: 员工列表/详情
- **Tasks**: 任务列表/详情
- **Budget**: 预算管理
- **Reports**: 绩效报表
- **Settings**: 设置

### 组件

- AppHeader: 顶部导航 + API Key 输入
- AppSidebar: 侧边栏导航

### 样式

- CSS Variables 主题系统
- 暗黑主题（默认）
- 响应式布局
